# Portal121Compat
Mc1.20.6 Paper插件 并不好用( ᖛ ̫ ᖛ )ʃ)
让 Paper 1.20.6 服务端尽量对齐 Minecraft 1.21 的地狱门创建行为。

`Portal121Compat` 是一个面向 Paper 1.20.6 的 Minecraft 服务端插件，用于处理或模拟 1.21 版本中与新地狱门创建相关的行为改动。它适合希望继续运行 1.20.6 服务端，同时又想减少玩家在跨版本体验中遇到差异的服务器。

## 插件定位

Minecraft 1.21 对地狱门创建和检测逻辑带来了一些行为变化。对于仍停留在 Paper 1.20.6 的服务器，这类变化可能导致玩家在搭建、激活或使用新地狱门时获得与新版不同的体验。

`Portal121Compat` 的目标是尽可能缩小这种差异，让服务器的地狱门行为更接近 1.21。

## 功能

- 适配 Paper 1.20.6 服务端。
- 处理与 1.21 新地狱门创建改动相关的逻辑。
- 减少玩家在不同 Minecraft 版本之间遇到的传送门行为差异。
- 适合生存服、半原版服、跨版本过渡服务器使用。
